# Bodega LVN



## Descrição

Bodega LVN é um aplicativo web responsivo feito em **HTML, CSS e JavaScript** que permite aos usuários (meus pais) gerenciar vendas e clientes fiados diretamente do celular, de forma prática e visual.

O app funciona offline e salva os dados localmente no dispositivo usando **LocalStorage**, permitindo registrar vendas, calcular totais, e manter histórico de clientes fiados.

---

## Funcionalidades

- Registrar vendas com produto, quantidade e preço.
- Cálculo automático do total de cada venda.
- Histórico de vendas do dia.
- Cadastro de clientes fiados e acompanhamento de débitos.
- Interface responsiva com tema escuro moderno, estilo caixa registradora.
- Funciona offline e pode ser adicionado à tela inicial do celular como app.
- Grátis, sem necessidade de login ou internet (exceto para acessar via GitHub Pages).

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage para armazenamento local
- Git/GitHub para versionamento e hospedagem (via GitHub Pages)

---

## Instalação

1. Abra o link do GitHub Pages no celular ou navegador:https://luanvs10.github.io/bodega-lvn/

2. Para instalar como app no celular:
   - **Android (Chrome)**: Menu → “Adicionar à tela inicial”
   - **iOS (Safari)**: Menu → “Adicionar à Tela de Início”

3. Comece a registrar vendas e clientes fiados. Todos os dados ficarão salvos localmente no dispositivo.

## Uso

1. Digite o nome do produto, quantidade e preço.
2. Clique em “Adicionar” para registrar a venda.
3. Vendas aparecem no **Histórico** e clientes fiados são listados.
4. Os dados ficam salvos no dispositivo mesmo fechando o app.
5. Para exportar dados, você pode implementar um botão de **Exportar CSV** (opcional).

## Observações

- O app é feito para uso pessoal da Bodega LVN.
- Por enquanto, não há login nem sincronização entre dispositivos diferentes.
- Futuras melhorias podem incluir gráficos de vendas, backup em nuvem e login de usuários.
